First time only: upload any placeholder version to GitHub → Vercel once, just to get your permanent link (e.g. https://your-project.vercel.app). That link never changes again.

Every time after that:
1. Paste that same link into the box above
2. Upload your picture
3. Tap Generate — downloads index.html + qr-code.png
4. Replace index.html in your GitHub repo (same filename) → push → same link updates automatically
5. Print/display qr-code.png — anyone scans it with their normal camera, it opens your link, then they point the camera at it again inside the page to see the picture pop up
